# 社員向け：新 PC キッティング手順

新しいパソコンを使えるようにする手順です。**所要時間 30〜60 分**。

途中で再起動が入ることがあるので、急がず進めてください。

---

## 事前準備（人が手でやること）

1. パソコンを起動し、画面の指示にしたがって **初期セットアップ** を完了
   - 言語: 日本語
   - キーボード: Microsoft IME
   - **ネットワーク**: 社内 Wi-Fi に接続（SSID とパスワードは岡島さんから受け取り）
   - **Microsoft アカウントは作らないでください**: 「ドメインに参加する」または「オフラインアカウント」を選ぶ
   - **ローカルアカウント名**: 岡島さんから渡される規約に従う
2. 岡島さんから次の情報を受け取る
   - 端末番号（例: `MSA-PC-042`）
   - 利用者名（あなたの名前）

---

## ステップ 1: PowerShell を「管理者として」起動

1. 画面左下の **検索** に `powershell` と入力
2. 「Windows PowerShell」の上で **右クリック** → 「管理者として実行」をクリック
3. 「このアプリがデバイスに変更を加えることを許可しますか？」→ **はい**

青い画面（ターミナル）が開きます。

## ステップ 2: 配布された 1 行コマンドを貼り付け

岡島さん or hide さんから受け取った **長いコマンド** を、青い画面の中に貼り付けます。

貼り付け方:
- コマンドをコピー
- 青い画面の中で **右クリック**（または `Ctrl + V`）
- そのまま **Enter** キーを押す

```powershell
# こんな感じの 1 行が貼られます（実際の URL は配布時に決まります）
Set-ExecutionPolicy Bypass -Scope Process -Force; irm https://github.com/.../bootstrap.ps1 | iex
```

## ステップ 3: 対話入力に答える

途中で次のことを聞かれます。岡島さんからの情報を見ながら入力してください。

| 聞かれる項目 | 入力例 | メモ |
|---|---|---|
| 端末番号 | `MSA-PC-042` | 岡島さん指定 |
| 利用者名 | `三浦 英直` | あなたのフルネーム |
| 部署 | `営業部` | 不明なら Enter キーだけ押せば OK |
| 設置場所 | `本社2F` | 不明なら Enter キーだけ押せば OK |

入力後、Enter キーを押すと次に進みます。

## ステップ 4: 待つ

アプリのインストールが順に流れます。**5〜30 分** ほどかかります。
画面に流れる文字を眺めながら、コーヒーでも飲んでてください。

## ステップ 5: 完了画面の確認

最後に **緑色の枠で「✅ キッティング Phase 1 完了」** と表示されれば成功です。

```
════════════════════════════════════════════
  ✅ キッティング Phase 1 完了
════════════════════════════════════════════
```

このあと、メール・VPN・共有フォルダ・プリンタの設定が **別途必要** です。
岡島さんに「キッティング完了したので、メールとプリンタの設定もお願いします」と連絡してください。

---

## ❌ 失敗した時

画面に **赤色の枠で「キッティング途中で失敗」** と表示されたら、慌てずに次の手順を:

1. 画面に書かれた **ログファイルの場所** をメモ（`C:\msa-kitting-logs\setup-...log`）
2. エクスプローラでそのファイルを開く
3. ファイル全文を選択（`Ctrl + A`）→ コピー（`Ctrl + C`）
4. **Claude のチャット画面**（https://claude.ai/）を開く
5. 次のメッセージを貼り付けて送信:

```
MSA キッティングで失敗しました。ログ全文を貼ります：

[ここに Ctrl + V でログ全文を貼り付ける]
```

Claude（CC）が原因を解析して、次にやるべきコマンドを返してくれます。
返ってきたコマンドを **PowerShell に貼って Enter** で続行できます。

## よくある質問

**Q. 途中で勝手に再起動した**
A. 正常です。再起動後、もう一度ステップ 1 から（PowerShell 管理者起動 → コマンド貼り付け）を行ってください。同じコマンドで OK です（重複インストールは自動でスキップされます）。

**Q. 「アクセスが拒否されました」と出る**
A. PowerShell が「管理者として」起動できていません。ステップ 1 をやり直してください。

**Q. インターネット繋がってる？**
A. Wi-Fi 接続を確認してください。Edge ブラウザで Google が開けるかチェック。

**Q. 何分くらい待てばいい？**
A. アプリ 5 本のインストールに 5〜30 分。回線速度や PC スペックによります。30 分超えたら一度 hide さん or 岡島さんに相談してください。
